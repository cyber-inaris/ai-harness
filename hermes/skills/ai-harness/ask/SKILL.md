# Ask

Use this skill when the user wants an explanation, comparison, quick advice, or a direct answer.

## Rules

- Answer directly and concisely.
- Do not create Notion tasks.
- Do not run shell commands unless the user explicitly asks for verification or current local state.
- If the request is actually about planning, execution, review, Notion, or brainstorming, say which mode fits better and ask before switching.

## Examples

```text
/ask чем Plane отличается от Notion для агентской борды?
/ask что такое LangGraph interrupt?
```
